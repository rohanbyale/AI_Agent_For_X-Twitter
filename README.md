</p>An intelligent AI-powered Twitter/X automation agent built using Google Gemini and the Model Context Protocol (MCP). It can generate engaging tweets, understand contextual information through MCP tools, and automatically publish posts to X.

---

✨ Features

- 🧠 AI-generated tweets using Gemini
- 🔗 Model Context Protocol (MCP) integration
- ✍️ Automated Twitter/X posting
- 📅 Scheduled content generation
- 🎯 Context-aware tweet creation
- 🔒 Secure API key management
- ⚡ Easy to configure and extend

---

🛠 Tech Stack

- Python
- Google Gemini API
- Model Context Protocol (MCP)
- Twitter/X API
- AsyncIO

pip install -r requirements.txt

GEMINI_API_KEY=your_api_key
